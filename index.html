Length in a string

Length in a string defines the length of the string .i.e, how many characters are present in the string

Example:
        var String ="Hello"
        var String2=String.length;
        console.log(String2)
     output:5

CharAt in string

CharAt returns the character of the specified index

Example:
        const StringChar="Hello"
        console.log(StringChar.charAt(1))
    output:e

indexOf in string

indexOf returns the character of the specified index

Example:
        const StringIndex="Hello"
        console.log(StringIndex.indexOf("e"))
    output:1

Replace in string

Replace in string replaces the string with new string

Example:
        const StringReplace="Hello"
        console.log(StringReplace.replace("Hello","world"))
        
    output:World

ReplaceAll in string

ReplaceAll in string replaces all the strings with new string that matches the selected string

Example:
        const StringReplace1="Hello is Full of Hello" 
        console.log(StringReplace.replaceAll("Hello","world"))
        
    output:world is Full of world

Split in a string

Split in a string is used to split the characters of the string

Example:
        const stringSplit="Hello"
        console.log(stringSplit.split(""))
      
        Output:["h","e","l","l","o"]

Reverse in a string

Reverse in a string is used to reverse the characters of the string

Example:
        const stringSplit1="Javascript";
        const stringsplit2=stringSplit1.split("")
        const stringreverse=stringsplit2.reverse();
        const stringjoin=stringreverse.join("")
        console.log(stringjoin)
    
        output:tpircsavaj

concat method is used to merge the two strings and returns the new string

toUpperCase in string

It is used to convert the lower case letters to upper case letters

Example:
        const StrConvert="hello";
        const StrResult=StrConvert.toUpperCase();
        console.log(StrResult)
    output:HELLO

toLowerCase in string

It is used to convert the upper case letters to lower case letters

Example:
        const StrConvert="HELLO";
        const StrResult=StrConvert.toLowerCase();
        console.log(StrResult)
    output:hello
